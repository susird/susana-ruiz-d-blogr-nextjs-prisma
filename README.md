## About this project
* 🏗️	 This application was build with Next.js
* 💎 The database schema was created with Prisma
* 🗄 The data was query and serve using graphQL
* 📨 The data were stored with PostgreSQL


### Installation
* git clone repo
* npm i

### Commands to start the application
* node src/server.js     
* npx prisma studio

### Build 
Run: npm run build
